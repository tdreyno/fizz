# DOM Listener Convenience Helper Mappings

These mappings are generated from TypeScript DOM event maps used by Fizz DOM builders. Each helper delegates to `.listen(eventName, toAction, options?)` with the same lifecycle and options behavior.

Naming rule:

- Prefix with `on`
- Camel-case the DOM event name
- Example: `mousedown` -> `onMouseDown`

The tables below list every helper currently exposed on each target builder.
